  - async cleanup() — Cleanup HTTP client and resources
  
Session Management:
  - async create_session(session_id, flow, working_dir, model, config)

  - async send_message(msg_type, session_id, flow, payload)
  
Prompt Streaming:

================================================================================
EVENT BUS API SUMMARY
================================================================================

Class: EventBus
  - __init__(queue_size) — Initialize with max queue size per subscriber

Pub/Sub:
  - async subscribe(workflow_id) → AsyncGenerator[WorkflowEvent, None]
    Creates per-workflow subscription channel
  
  - async publish(workflow_id, event) → None
    Non-blocking publish; drops on queue full with warning
  
  - async unsubscribe(workflow_id, subscriber_id) → None
    Manual unsubscribe (auto-called on generator close)

Introspection:
  - async get_subscriber_count(workflow_id) → int
  - async get_stats() → dict (workflows, total_subscribers)

Module-level Singleton:
  - get_event_bus() → EventBus — Get global instance
  - set_event_bus(bus) → None — Set global instance (testing)

================================================================================
EVENT TYPES DEFINED
================================================================================

All events inherit from WorkflowEvent with:
  - type: str (discriminator)
  - workflow_id: str
  - timestamp: datetime (UTC, RFC 3339)

Specific Event Types:

1. WorkflowStateChangedEvent
   Fields: from_state, to_state, trigger, review_cycle, metadata

2. AgentStreamStartEvent
   Fields: role, agent, session_id

3. AgentStreamDeltaEvent
   Fields: role, content_type (text|markdown|code|json), content

4. AgentStreamEndEvent
   Fields: role, finish_reason (stop|length|error|tool_use|cancelled)

5. AgentStatusEvent
   Fields: role, status (thinking|tool_use|idle|error), details

6. AgentToolUseEvent
   Fields: role, tool_name, input (dict), tool_use_id

7. ReviewCommentsEvent
   Fields: cycle (int), comments (list), consensus (bool), agent

8. ResolutionMergedEvent
   Fields: cycle (int), resolutions (list)

9. ReportUpdatedEvent
   Fields: version (int), path (str), format (markdown|html|pdf)

10. UserChatResponseEvent
    Fields: content (str), streaming (bool)

11. WorkflowCompletedEvent
    Fields: final_report_path (str), summary (str)

12. WorkflowErrorEvent
    Fields: code (str), message (str), recoverable (bool), details (dict)

13. EventType (enum)
    All event type string constants

14. Utility Function
    serialize_event(event) → dict — Convert to JSON-serializable dict
